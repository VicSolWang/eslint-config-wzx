The custom eslint rules based on airbnb-base

# Installation

    npm install --save-dev eslint-config-wzx

# Usage

Add eslint-config-wzx to the "extends" in your .eslintrc.* file.

    {
        "extends": "eslint-config-wzx"
    }
}
